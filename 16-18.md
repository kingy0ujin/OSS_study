# 16. 원격 저장소 복제 clone
## 클론(clone)
- 원격 저장소(깃허브, 비트버킷)를 지역 저장소에 복제
- 공개된 저장소는 누구나 가능


## 원격 저장소 생성, 복제
- 깃허브 원격 저장소 생성 -> 저장소 이름 git clone, http 주소 복사
  ```
  $ git clone [복사된 주소] -> 원격저장소와 동일한 이름으로 복제
  $ git clone [복사된 주소] [새로운 폴더명] -> 하부 폴더 [새로운 폴더명] 이름으로 복제
  $ git clone [복사된 주소]. -> 현재 폴더에 바로 복제
  ```
